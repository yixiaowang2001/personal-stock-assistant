# 日常启动指南

本文档说明**每次使用**时如何启动程序（非首次安装/初始化）。

---

## 前提

- 已完成《安装指南.md》中的环境安装与首次初始化（MongoDB 用户、Redis 密码、`.env`、数据库初始化脚本等）。
- macOS 下 MongoDB、Redis 通过 Homebrew 安装并已配置好。

---

## 启动步骤

### 1. 确保 MongoDB 和 Redis 在运行

```bash
# 若未启动，执行：
brew services start mongodb/brew/mongodb-community
brew services start redis
```

检查是否已在运行：

```bash
brew services list | grep -E "mongodb|redis"
```

### 2. 启动后端

在**项目根目录**打开终端：

```bash
cd /Users/wangyixiao/Desktop/Files/Projects/personal-stock-assistant
source venv/bin/activate
python -m app
```

看到 `Uvicorn running on http://0.0.0.0:8000` 即表示后端已启动，API 地址：<http://localhost:8000>。

### 3. 启动前端

**新开一个终端**，执行：

```bash
cd /Users/wangyixiao/Desktop/Files/Projects/personal-stock-assistant/frontend
npm run dev
```

看到 Vite 提示的本地地址（一般为 <http://localhost:3000>）即表示前端已启动。

### 4. 使用

- 在浏览器打开前端地址（如 http://localhost:3000）。
- 默认登录：用户名 `admin`，密码 `admin123`。

---

## 关闭

- 在后端终端按 `Ctrl+C` 停止后端。
- 在前端终端按 `Ctrl+C` 停止前端。
- MongoDB 和 Redis 会随系统/Homebrew 常驻，无需每次关闭；若要停止：

  ```bash
  brew services stop mongodb/brew/mongodb-community
  brew services stop redis
  ```

---

## 简要速查

| 步骤 | 命令 |
|------|------|
| 启动数据库 | `brew services start mongodb/brew/mongodb-community`、`brew services start redis` |
| 启动后端 | 项目根目录 → `source venv/bin/activate` → `python -m app` |
| 启动前端 | `cd frontend` → `npm run dev` |
| 后端地址 | http://localhost:8000 |
| 前端地址 | http://localhost:3000（以终端输出为准） |
